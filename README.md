# hpc-project

## How to run:

Prerequirements:
- Python 3.7 or higher
- g++ compiler
- OpenMP for C/C++


First run command for package installation:
```
pip install mathplotlib numpy
```

Run command for shell script:
```
chmod +x run.sh
```

Run solution:
```
./run.sh <dimension>
```
where dimension is n for maze nxn
